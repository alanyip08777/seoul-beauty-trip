# 2026 首爾 5天4夜｜變美 × 購物 × 出片

## 功能
- 前台行程展示
- 後台新增 / 刪除行程
- 後台新增 / 刪除購物清單
- 三麗鷗 / Pochacco 重點清單
- 美妝藥妝分類
- localStorage 自動儲存

## 本地啟動
```bash
npm install
npm run dev
```

## Vercel 部署
1. 把整個專案上傳到 GitHub repo
2. 回 Vercel → Add New Project
3. Import Git Repository
4. Framework Preset 選 Vite
5. Build Command: `npm run build`
6. Output Directory: `dist`
7. Deploy
